# skimr — site

Astro static site for skimr. Content is generated by the Python pipeline into
`src/content/items/` and rendered as a static site.

## Local development

```bash
npm ci
npm run dev
```

The dev server reads from `src/content/items/`. The pipeline writes to that
directory. Run `python -m pipeline.main` from the repo root to populate it.

## Build

```bash
npm run build
```

Output goes to `dist/`.

## Configuration

Edit `astro.config.mjs` and set `site` to your GitHub Pages URL. If you deploy
to `<username>.github.io/skimr/`, leave `base: '/skimr'`. If you deploy to a
custom domain or `<username>.github.io/` root, remove `base`.
